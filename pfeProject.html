Real time embedded system for monitoring the temperature inside refrigerated cabins

In this work, we focus on the logistics industry and propose an innovative solution for developing a real-time IoT system to track the temperature inside refrigerated containers and monitor the opening of doors in utility trucks. The hardware component involves designing and implementing a real-time tracking embedded system using Quectel MC60. The software component consists in developing a web application developped with Flutter and SpringBoot for data analysis, visualization, remote control, and receiving alerts from the IoT system. The proposed solution utilizes development software and dedicated communication protocols for IoT systems like MQTT and BLE.

Technologies: GSM/GPRS module Quectel MC60, OpenCPU, BLE module HM10, embeddedC, Flutter, SpringBoot, MQTT, Mysql.
